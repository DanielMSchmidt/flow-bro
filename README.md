# Flow-Bro

![Icon](https://i.imgflip.com/1mdkbx.jpg)

## Installation

```
npm install -g flow-bro
yarn global add flow-bro
```

## Usage

### Commands
#### `flow-bro <number>`

It gets you the most untyped components in the current directory.
Example: `flow-bro get-untyped 25`
